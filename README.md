# ファイル作成ロボット

## GOAL
ファイル名を記入すると、  

①ファイル名と同じ名前を持つフォルダ  
②初期設定が既に記入されていて、入力したファイル名をもうファイル  
③ReadMeのテンプレートファイル  
の3つが入ったフォルダが作成される。 


また、フォルダ作成が完了した際、作成完了というログが画面上に表示されるようにする。
